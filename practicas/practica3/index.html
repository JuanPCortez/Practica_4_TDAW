¿Qué es el desarrollo Web?

El desarrollo web es un proceso complejo que implica la creación de productos digitales en línea. También es un término que define la creación de sitios web para Internet o una intranet. Para conseguirlo se hace uso de tecnologías de software del lado del servidor y del cliente que involucran una combinación de procesos de base de datos con el uso de un navegador web a fin de realizar determinadas tareas o mostrar información.

  1. HTML

    2. El Lenguaje de Marcado de Hipertexto (del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Algunas de las etiquetas más importantes de nuestro proyecto son:

    • a

      • Abre un enlace

    • img

      • Inserta una imagen

    • p

      • Sirve para insertar parrafos

  2. CSS

    3. Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) es el lenguaje de estilos utilizado para describir la presentación de documentos HTML, XML, entre otros. CSS describe como debe ser renderizado el elemento estructurado en la pantalla. Algunas de las propiedades más importantes de nuestro proyecto son:

    • Color

      • Pone el elemento deseado de un color que el usuario define

    • Backround color

      • pone el fondo del elemento deseado de un color que el usuario elige

    • Font-size

      • Pone la fuente de un texto del tamaño que el usuario desea
